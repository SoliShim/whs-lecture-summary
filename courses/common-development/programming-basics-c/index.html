작성 기준

  • 외부 자료나 인터넷 이미지는 사용하지 않았다. 사용자가 요청한 “STT 자료만” 조건을 지키기 위해 강의 원문에 근거한 설명만 넣었다.
  • STT의 자동 전사 오류로 보이는 단어는 문맥상 의미가 분명한 경우에만 학습 가능한 표현으로 정리했다.
  • 1-1과 1-2는 STT 내용이 동일하므로, 1-2 페이지에는 중복 전사 사실을 명시했다.
